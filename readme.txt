Adds / removes Honey Pot links to / from existing posts / pages.

== Description ==

Project Honey Pot makes possible to install traps against comment spammers 
and e-mail harvesters. These traps have to be activated by adding special 
links to posts / pages. This plugin automates the process by inserting such 
links to all existing posts / pages. 

== Installation ==

1. Upload `fpw-honey-pot-links` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Select Dashboard's Settings -> FPW Honey Pot Links to modify settings
